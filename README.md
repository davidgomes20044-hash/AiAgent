# AgentOS — AI Automation Agent

Autonomous AI agent that executes complex tasks in natural language using the Anthropic Claude API.

## Demo
> Open `agent.html` in your browser, paste your API key, write a task — done.

## Features
- Breaks down complex tasks into sequential steps automatically
- Real-time execution progress
- Generates detailed analysis and full reports
- Zero setup — no backend, no server, no dependencies

## Stack
- Pure HTML + CSS + JavaScript (zero frameworks)
- Claude API (claude-sonnet) called directly from the browser

## How to use
1. Open `agent.html` in your browser
2. Enter your [Anthropic API key](https://console.anthropic.com/)
3. Write a task in natural language
4. Click **Execute Task**
